## Glossary

- AI Detection: Process of identifying AI-generated text using GPTZero API
- Assessment Criteria: Standards by which student work is evaluated
- Class Code: Unique 6-digit code for student enrollment
- Content Item: Generic term for worksheets, podcasts, or mind maps
- Curriculum Alignment: How content matches UK National Curriculum standards
- Edge Function: Serverless function running on Supabase (Deno runtime)
- Learning Objective: Specific, measurable goal for student learning
- Mind Map: Visual diagram organizing information hierarchically
- Research Quality Score: 0–100 rating of research reliability and relevance
- Row-Level Security (RLS): Database security ensuring users only access their own data
- Share Code: Unique 8-character code for public content access
- Worksheet Submission: Student's completed worksheet responses
- Year Group: UK education level (Reception through Year 13)
