## 1. Introduction

### What is Reducation?

Reducation is a comprehensive, AI-powered educational platform designed specifically for UK educators. It provides a complete suite of tools for lesson planning, content creation, research, assessment, and class management, all enhanced with cutting-edge artificial intelligence.

### Key Features

- AI-Assisted Lesson Planning: Generate complete, curriculum-aligned lesson plans in minutes
- Interactive Content Creation: Create worksheets, podcasts, and mind maps with AI assistance
- Fact-Checked Research: Conduct research with source validation and quality scoring
- Class Management: Manage students, assignments, and track progress
- Assessment Tools: AI-powered grading and feedback generation
- Public Sharing: Share worksheets and content with students via unique links
- Analytics & Reports: Track performance and engagement metrics

### Target Users

- Primary school teachers (Reception - Year 6)
- Secondary school teachers (Year 7 - Year 13)
- Subject coordinators
- Head teachers and administrators
- UK-based educational institutions

### Technology Overview

Reducation is built with modern web technologies:
- Frontend: React 18, TypeScript, Tailwind CSS
- Backend: Supabase (PostgreSQL, Auth, Edge Functions)
- AI Services: OpenRouter (Claude & Gemini), ElevenLabs, Fal.ai, GPTZero
